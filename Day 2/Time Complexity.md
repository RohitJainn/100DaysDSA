### 2
    void fun(int n){
    i=1, s=1;
    while(s<=n){
        i++;
        s=s+i;
        print("a")
        }
    }
- Time complexity = O(sqrt(n))
### if k is the total number of iterations taken,then the while loop terminates if 1+2+3+...+k = [k(k+1)/2] > n. So k = O(sqrt(n))
